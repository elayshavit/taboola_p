# Logo File Naming

Place your company logos in this folder with these exact filenames (slug-based).
Update `config/brand.ts` logoSrc for each company if you use a different extension.

| Original Name (example) | Required Filename | Path |
|------------------------|-------------------|------|
| Taboola.png            | taboola.png       | /logos/taboola.png |
| Teads.png              | teads.png         | /logos/teads.png |
| The Trade Desk.png     | the-trade-desk.png| /logos/the-trade-desk.png |
| Simpli.fi.png          | simpli-fi.png     | /logos/simpli-fi.png |

Supported extensions: .png, .svg, .webp

Placeholder SVGs (monogram style) are included. Replace with your real logos and update
`config/brand.ts` logoSrc if using .png or .webp.
